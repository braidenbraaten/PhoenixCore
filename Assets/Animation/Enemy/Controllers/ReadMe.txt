DIVE BOMBER CONTROLLER INFORMATION

			--STATES--

-Idle: Will be used for the plane's idle animation.

-AcquiringTarget:If the plane sees the player / building that it is targeting
-DiveAtTarget: After it has aquired the target, it will procede to dive at it

DEATH STATES
-HitPlayer (if it succeeds in hitting the player)
-CrashedState (if it succeeds in hitting the target building)
-ShotDown (if the player shoots down the plane before it reaches it's target)
